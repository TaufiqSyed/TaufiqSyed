About me

Computer Science is my passion and I have been practicing OOP, front end development and coding interview questions on my own time. I know that there is a much, much longer route to mastery, and am therefore always ready to learn the next thing.

I've learnt a bit of front end development from resources such as YouTube, Stack Overflow, and the MDN Web Docs. While I prefer working with console applications, designing a webpage through HTML and CSS creates a much more tangible end product, and I would love for an oppurtunity to build up my skillset.

As for algorithm designing, I have been practicing coding interview questions to much success. My preferred language is Python 3 but I have dabbled in VB.NET, Object Pascal / Delphi, C#, C++ and Javascript, from most proficient to least. My greatest flaw is obsessing over small details, such as making a program just a couple lines less redundant, instead of better spending my time.
